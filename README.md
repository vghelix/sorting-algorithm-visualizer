# sorting-algorithm-visualizer
This Java Algorithm Visualizer is a desktop application built using Java Swing that renders the execution of sorting algorithms
<img width="1916" height="1012" alt="image" src="https://github.com/user-attachments/assets/7ed06f62-40ab-4d75-a6ea-97eb055dbe40" >
The project utilizes concurrency by running the sorting logic on a separate thread, ensuring the GUI remains responsive while pausing between steps to allow the user to visually understand the mechanics of the algorithm.
